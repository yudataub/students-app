📋 הוראות הגדרה (Firebase חינמי)
  1. כנס ל-console.firebase.google.com עם חשבון Google
  2. לחץ "צור פרויקט" → תן שם → המשך
  3. בפרויקט: לחץ </> (Web App) → תן שם → Register app
  4. העתק את כל הטקסט של firebaseConfig = {'{'}...{'}'} — רק את החלק בסוגריים המסולסלים
  5. Authentication → Sign-in method → Google → הפעל ✓
  6. Storage → Get started → Production mode → Done
  7. Storage → Rules → שנה ל:
    allow read, write: if request.auth != null && request.auth.uid == userId;
    (שמור את הקובץ המלא שיופיע בשלב זה ← הצג rules מלאים)
  8. הדבק את הקונפיגורציה בתיבה למטה
